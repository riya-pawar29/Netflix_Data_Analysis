# Netflix Movies & TV Shows Data Analysis using Python

## 📌 Project Overview
This project analyzes the Netflix Movies & TV Shows dataset using Python. The goal is to clean the data, perform exploratory data analysis (EDA), and visualize important insights using different charts.

## 📂 Dataset
- Netflix Titles Dataset
- Source: Kaggle

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Visualizations
- Histogram
- Scatter Plot
- Box Plot
- Heatmap
- Count Plot

## 📈 Key Insights
- Movies are more than TV Shows.
- Most content is around 90–100 minutes.
- The majority of content was released after 2015.
- Some columns contained missing values which were cleaned before analysis.
